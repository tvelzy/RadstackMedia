Contact list

  

Here you can see a list of contacts available on the current site and you can also create new contacts; please note that contacts are objects covering various roles which a person has in the CMS system, e.g. user, customer, subscriber etc.

 

If global contacts are allowed for the current site (Site Manager -> Settings -> On-line marketing -> Contact management -> Global data & merging), the Site drop-down list is visible enabling you to view current site contacts, global contacts or a mix of global and current site contacts. However, if global contacts are not allowed for the current site, only contacts created on the current site can be viewed. If available, the Site drop-down list offers you the following options:

 

(global and this site)

Both global and current site contacts are listed. The option does not allow to create a new contact.

(global)

Only global contacts are listed. The option allows to create a new global contact by clicking New contact.

current site

Only current site contacts are listed. The option allows to create a new site contact by clicking New contact.

 

 

 

Please note

 

If you are accessing this page from Site Manager -> Tools -> Contact management, the Site drop-down list allows you to work with all available sites.

 

 

You may need to display only contacts matching certain search criteria (e.g. first name, last name, e-mail etc.). To do so, choose either the simplified or advanced filter, enter the required values and click Search.

 

Individual contacts can be Edited () or Deleted () and you can also perform Other actions such as Export, Backup, Restore or Clone. The displayed list can be sorted by contact first name, last name, e-mail, contact status and country. After clicking the  icon in the header of the Actions column, a context menu is displayed, offering options for export of listed data to various types of files.

 

Besides managing individual contacts, you can perform certain contact management tasks at once, specifically adding contacts to a contact group, deleting and merging contacts or changing the contact status. Choose either Selected contacts or All contacts from the drop-down list and click OK.
